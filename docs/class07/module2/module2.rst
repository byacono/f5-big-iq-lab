Module 2: Network Tools and Logs
================================

In this module, we will explore using BIG-IQ to centrally view AFM logs and leverage built-in tools to evaluate and monitor firewall policy.

Specifically, we will work with the following BIG-IQ features:
 - Multi Device Packet Tester
 - Support AFM log aggregation and Visibility
 - Ability to rename an object related to firewall
 - GUI support device dos multi-editing function
